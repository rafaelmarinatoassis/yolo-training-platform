Como Interpretar as Metricas
Training Loss (Erro)

O que e: Mede o quanto o modelo esta errando durante o treinamento.

Como ler:

  • Diminuindo = Modelo esta aprendendo
  • Estavel = Modelo pode ter convergido
  • Aumentando = Possivel overfitting
Meta: Valores menores sao melhores (idealmente menor que 1.0)
mAP50 (Precisao Principal)

O que e: Mede a precisao geral do modelo para detectar objetos.

Como ler:

  • > 50% = Modelo muito bom
  • 30-50% = Modelo razoavel
  • < 30% = Precisa melhorar
Meta: Valores maiores sao melhores (maximo 100%)
Precisao (%)

O que e: Dos objetos que o modelo detectou, quantos estavam corretos?

Como ler:

  • Alta precisao = Poucas deteccoes falsas
  • Baixa precisao = Muitos falsos positivos
Exemplo: 80% = de 100 deteccoes, 80 estavam corretas
Recall (Sensibilidade)

O que e: Dos objetos que existiam na imagem, quantos o modelo conseguiu encontrar?

Como ler:

  • Alto recall = Encontra quase tudo
  • Baixo recall = Perde muitos objetos
Exemplo: 70% = de 100 objetos reais, encontrou 70
Dica de Interpretacao:
Modelo Ideal:
Loss baixo, mAP50 alto, Precisao e Recall equilibrados
Trade-off:
As vezes precisao alta = recall baixo e vice-versa
Progresso:
Observe as tendencias ao longo das epocas
